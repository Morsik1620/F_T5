HTML & CSS

Достижения и навыки в использовании тегов HTML и стилизации CSS

Теги HTML

Пройдя обучение второго занятия, были изучены материалы по использованию тегов HTML и стилизации страницы через файл CSS. Для себя выделил такие основные теги как:

  • <html> - контейнер для содержимого страницы;
  • <head> - содержит служебную информацию;
  • <meta> - Определяет метаданные о HTML-документе
  • <title> - Определяет заголовок для документа
  • <link> - Определяет взаимосвязь между документом и внешним ресурсом
  • <body> - содержит контент - всё, что отображается в браузере;
  • <h1>-<h6> - Определяет заголовки HTML;
  • <p> - создаёт абзацы;
  • <ul> - создаёт неупорядоченный список;
  • <li> - добавляются элементы списка;
  • <ol> - создаёт упорядоченный список;
  • <div> - создаёт секцию в документе;
  • <table> - создаёт таблицу;
  • <tr> - определяет строку в таблице;
  • <a> - используется для создания ссылок;
  • <button> - создаёт кнопку.

Стилизация CSS

В ходе обучения открыл для себя возможности файлов CSS. Они мне показались интересными, удобными и мощными инструментами в плане стилизации HTML страниц. Для себя выделил следующие возможности:

  1. Color: устанавливает цвет текста элемента.
  2. Font-family: устанавливает шрифт для текста элемента.
  3. Font size: устанавливает размер текста элемента.
  4. Background color: устанавливает цвет фона элемента
  5. Width and height: устанавливает ширину и высоту элемента.
  6. Text alignment: устанавливает выравнивание текста внутри элемента.

В качестве шпаргалки, в свой блог включил ссылки на ресурсы раскрывающие темы HTML и CSS.
